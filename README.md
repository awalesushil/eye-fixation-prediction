# Eye Fixation Prediction

Predicting Eye Fixation map given an image (Computer Vision 2 Project)

## Install dependencies

```
python3 -m venv .env
source .env/bin/activate
pip3 install requirements.txt
```

## Unzip data

```
python3 unzip.py
```

## Train

```
python3 train.py
```

## Predictions

```
python3 predictions.py
```
